# MSCS532_QUICK_SORTING
Quicksort Implementation (Assignment 5)
Overview

This project implements the Quicksort algorithm in two ways:
A standard version using a fixed pivot
A randomized version using a random pivot
The goal is to understand how pivot selection affects performance.

## How to Run

Run the files using Python:

python deterministic_quicksort.py
python quicksort_randomized.py

## Summary

Both implementations sort the array correctly.
The randomized version helps avoid worst-case scenarios and gives more consistent performance.
